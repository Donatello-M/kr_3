### Задание 3 

Что именно будет выведено на экран и почему? Как функция findMax взаимодействует с массивом в main? Как через указатель p изменить значение прямо в исходном массиве?
```c
int* findMax(int* arr, int size) {
   int* maxPtr = &arr[0];
   for (int i = 1; i < size; i++) {
       if (arr[i] > *maxPtr) maxPtr = &arr[i];
   }
   return maxPtr;
};

int main() {
   int data[] = {10, 40, 20};
   int* p = findMax(data, 3);
   printf("%d", *p);
   return 0;
};
```

### Задание 4

Зачем здесь используются «двойные указатели» (int**)? Что произойдет, если в main передать просто pMin, а не его адрес &pMin?
```c
void getExtrema(int* arr, int size, int** pMin, int** pMax) {
   *pMin = &arr[0];
   *pMax = &arr[0];
   for (int i = 1; i < size; i++) {
       if (arr[i] < **pMin) *pMin = &arr[i];
       if (arr[i] > **pMax) *pMax = &arr[i];
   }
};
```
