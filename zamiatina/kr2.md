### Задание 3

В приведенном ниже коде допущена ошибка. Найдите её, объясните, к чему она приведет, и исправьте код так, чтобы функция возвращала адрес максимума в столбце матрицы.
```c
int* maxInColumn(int rows, int cols, int mat[rows][cols], int c) {
   int maxVal = mat[0][c];
   for (int i = 1; i < rows; i++) {
       if (mat[i][c] > maxVal) maxVal = mat[i][c];
   }
   return &maxVal;
};
```

### Задание 4

Как работает сравнение указателей p1 < p2? Почему это условие позволяет определить порядок элементов в памяти и корректно задать границы цикла?
```c
int sumBetween(int* p1, int* p2) {
   int sum = 0;
   int* start = (p1 < p2) ? p1 : p2;
   int* end = (p1 < p2) ? p2 : p1;
  
   for (int* curr = start + 1; curr < end; curr++) {
       sum += *curr;
   }
   return sum;
};
```
