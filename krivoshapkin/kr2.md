### Задание 3

Почему без использования указателей значения переменных x и y в main не изменятся? Опишите пошагово процесс обмена данными в памяти.
```c
void swap(int* a, int* b) {
   int temp = *a;
   *a = *b;
   *b = temp;
};


int main() {
   int x = 5, y = 10;
   swap(&x, &y);
   return 0;
};
```


### Задание 4

В приведенном ниже коде допущена ошибка. Найдите ее и сделайте так, чтобы Функция искала максимум и увеличивала его значение в массиве на 1.
```c
void findAndIncrementMax(int* arr, int size) {
   int* pMax = &arr[0];
   for (int i = 1; i < size; i++) {
       if (arr[i] > *pMax) {
           pMax = &arr[i];
       }
   }
  
   *pMax++;
};
```
