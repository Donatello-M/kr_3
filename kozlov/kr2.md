### Задание 3 

Почему возвращаемый тип функции — int*? Что будет содержать переменная minPtr в main после завершения вызова?
```c
int* getMinPtr(int* data, int n) {
   int* min = data;
   for (int i = 1; i < n; i++) {
       if (*(data + i) < *min) min = data + i;
   }
   return min;
};
```

### Задание 4

Зачем здесь нужен массив указателей refs[]? Как через этот массив можно точечно изменить значения в исходном массиве arr внутри функции main?
```c
void collectZeros(int* arr, int size, int* refs[], int* count) {
   *count = 0;
   for (int i = 0; i < size; i++) {
       if (arr[i] == 0) {
           refs[*count] = &arr[i];
           (*count)++;
       }
   }
};
```
